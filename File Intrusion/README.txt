################################    Breach Codes   ####################################
1. 001 - File number is less than that in database
2. 002 - Some filenames does not match that in database
3. 003 - Some file extensions does not match that in database
4. 004 - Some file hashes does not match that in database


################################    How to Use   ####################################
1. Files that  you want to monitor its intergrity should be copied or cut into the  
"Safe" folder
Or 
You can Add files to the "Safe" folder by opening the app and pressing the add button
You can Remove file the same way too.

2. Every times files are added or removed from the "Safe" folder press the SET Button
to reset the database. So as not to mistakenly flag a file breach

3. The 'Check' Button is used to check the files in the "Safe" Folder


Note: You Want to monitor a Folder intergrity pls compressed the folder before adding it
to the "Safe" folder
